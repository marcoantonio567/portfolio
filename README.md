# Marco Antonio | Portfolio

Personal portfolio built with HTML, CSS, and JavaScript to present a professional profile, technologies, selected projects, and contact options.

## Overview

This website highlights Marco Antonio's work as a developer, with a focus on web systems, mobile apps, automation, data analysis, and artificial intelligence projects.

## Features

- Responsive layout for desktop, tablet, and mobile devices
- Section navigation: About, Projects, Services, and Contact
- Language switch between Portuguese and English
- Scroll reveal animations
- Interactive project cards
- Links to GitHub, repositories, email, and LinkedIn

## Technologies

- HTML5
- CSS3
- JavaScript
- Google Fonts: Inter and JetBrains Mono

## Project Structure

```text
.
├── index.html
├── styles.css
├── script.js
└── README.md
```

## How To Run

This is a static project, so it does not require dependency installation.

1. Download or clone the repository.
2. Open the `index.html` file in your browser.

You can also use an extension like Live Server in VS Code to preview the website with automatic reload.

## Customization

To customize the portfolio:

- Edit texts, links, and projects in `index.html`
- Adjust colors, spacing, and responsiveness in `styles.css`
- Update animations and language switching in `script.js`
